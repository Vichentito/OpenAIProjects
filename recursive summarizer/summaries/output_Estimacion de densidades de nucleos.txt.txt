Esta publicación se centra en la estimación de densidades basadas en núcleos, una técnica no paramétrica útil para estimar una función de densidad a partir de un conjunto de variables aleatorias. El método implica minimizar una función de costo, R(f), y un estimador insesgado de este valor es: R(f)= P-f f2. Además, el método de validación cruzada por máxima verosimilitud maximiza log f_i (Xi). Se compara la estimación obtenida con la ventana referida a la distribución normal, y la ventana obtenida por sustitución. También se compara el uso de distintas ventanas en el caso gaussiano, y se aplica el método para dimensiones mayores a 2. Finalmente, se discute el uso de núcleos simétricos con una sola ventana, y el uso de núcleos asimétricos para datos asimétricos.

Podemos usar la ventana (10) directamente. Si estamos usando el estimador (8) con el núcleo simétrico, en lugar de usar HOPT, usaremos OHOPT, donde O2 es un estimador de la varianza marginal promedio. Para el método de validación cruzada, se usa la fórmula (6), donde K(x) = kK (e/h). El método de sustitución también se traslada a R. Algunos paquetes utilizan este método para estimar por separado distintas ventanas para las distintas dimensiones. Debido a las formulas de varianza y ventana óptima, la varianza del estimador tiende a cero como L, haciendo necesario un número de orden exponencial para mantener una varianza dada. El estimador más sencillo y antiguo es el histograma. El estimador "naive" calcula la densidad con No. de X en [x h, x + h] / 2nh. Estimamos la densidad conjunta para los datos del geyser Old Faithful con histogramas para 10, 20 y 35 intervalos respectivamente, y estimaciones naive para datos con distribución normal típica para 50 y 500 datos respectivamente.

El estimador naive se define como una función w definida como: w(x) = 22 si |x| < 1, 0 si no. Esta función puede ser generalizada sustituyendo w por una función K tal que fkK(x) = 1. El Error Cuadrado Medio (ECM) es una medida global del error dada por la varianza de f(x) y el sesgo del estimador. Existe un compromiso entre el sesgo y la varianza para h, de manera que cuando h es muy pequeña el sesgo disminuye pero aumenta la varianza. Para encontrar la ventana óptima se minimiza el ECMI con respecto a h, obteniéndose la ecuación (3). Existen además métodos de selección de ventanas como Silverman's "rule of thumb", los métodos de sustitución (Plug-in) de Park y Marron, y de Sheater y Jones, así como el método de validación cruzada basado en mínimos cuadrados.
